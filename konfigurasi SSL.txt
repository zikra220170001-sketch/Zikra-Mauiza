Konfigurasi Keamanan SSL
Langkah 1: Pastikan file openssl.cnf ada
1.	Pastikan di C:\xampp\apache\conf\ ada file: openssl.cnf
2.	Kalau tidak ada, salin dari: C:\xampp\apache\bin\openssl.cnf
3.	ke folder: C:\xampp\apache\conf\

Langkah 2: Buka CMD sebagai Administrator
Masuk ke folder openssl: cd C:\xampp\apache\bin

Langkah 3: Buat Private Key
Buat folder ssl:  mkdir C:\xampp\apache\conf\ssl\
openssl genrsa -out C:\xampp\apache\conf\ssl\localhost.key 2048

Langkah 4: Buat Certificate Signing Request (CSR)
openssl req -new -key C:\xampp\apache\conf\ssl\localhost.key –out C:\xampp\apache\conf\ssl\localhost.csr -config C:\xampp\apache\conf\openssl.cnf
Saat muncul prompt Country Name (2 letter code) [AU]:
Isi aja bebas seperti contoh:
Country Name (2 letter code) [AU]: ID
State or Province Name: Aceh
Locality Name: Lhokseumawe
Organization Name: CloudSecure Project
Common Name: localhost
Email Address: admin@localhost

Langkah 5: Buat Sertifikat SSL (.crt)
openssl x509 -req -days 365 -in C:\xampp\apache\conf\ssl\localhost.csr –signkey C:\xampp\apache\conf\ssl\localhost.key -out C:\xampp\apache\conf\ssl\localhost.crt
Kalau berhasil, di folder: C:\xampp\apache\conf\ssl\ akan muncul 3 file:
localhost.key
localhost.csr
localhost.crt

Langkah 6: Edit File Konfigurasi Apache
Buka file: C:\xampp\apache\conf\extra\httpd-ssl.conf
Cari baris yang aktif (Tidak dikomentari #):
SSLCertificateFile
SSLCertificateKeyFile
Pastikan diarahkan ke file kamu:
SSLCertificateFile "C:/xampp/apache/conf/ssl/localhost.crt"
SSLCertificateKeyFile "C:/xampp/apache/conf/ssl/localhost.key"

Langkah 7: Aktifkan Modul SSL di Apache
Buka file: C:\xampp\apache\conf\httpd.conf
Pastikan tiga baris ini tidak dikomentari (hapus tanda #):
LoadModule ssl_module modules/mod_ssl.so
Include conf/extra/httpd-vhosts.conf
Include conf/extra/httpd-ssl.conf 

Langkah 8: Cek Apache aktif di por 443
Buka Command Prompt (CMD): netstat -an | find "443"

Langkah 9: Instal mkcert (tool membuat sertifikat HTTPS lokal yang terpercaya).
Download mkcert dari halaman GitHub: https://github.com/FiloSottile/mkcert/releases
Ubah namanya jadi mkcert.exe
Pindahkan ke folder: C:\Windows\System32

Langkah 10: Instal Root Certificate (CA lokal)
Buka Command Prompt (CMD) sebagai Administrator
Jalankan: C:\Windows\System32\mkcert.exe –version
Instal root CA: C:\Windows\System32\mkcert.exe -install

Langkah 11: Buat Sertifikat untuk localhost
Buat sertifikat SSL khusus domain localhost: C:\Windows\System32\mkcert.exe localhost
Akan muncul dua file: localhost.pem & localhost-key.pem

Langkah 12: Mengaktifkan SSL
Pindahkan dua file tadi ke: C:\xampp\apache\conf\ssl\
Edit file konfigurasi Apache: C:\xampp\apache\conf\extra\httpd-vhosts.conf
Tambahkan di bagian bawah:
<VirtualHost *:443>
    ServerName localhost
    DocumentRoot "C:/xampp/htdocs"
    SSLEngine on
    SSLCertificateFile "C:/xampp/apache/conf/ssl/localhost.pem"
    SSLCertificateKeyFile "C:/xampp/apache/conf/ssl/localhost-key.pem"
</VirtualHost>

Langkah 13: Akses Website Aman
Sekarang buka di browser: https://localhost/cloudsecure


Hasil Pengujian Keamanan SSL dengan OpenSSL s_client
Buka Command Prompt (CMD) sebagai Administrator
Masuk ke folder openssl: cd C:\xampp\apache\bin
Jalankan perintah pengujian koneksi SSL: openssl s_client -connect localhost:443

Artinya:
TLS v1.3 digunakan → protokol terenkripsi terbaru (aman).
Cipher kuat seperti AES_256_GCM_SHA384 digunakan → enkripsi modern.
Verify return code: 0 (ok) → sertifikat valid dan dipercaya sistem.
